La Balise <A>



La balise <a> définit un lien hypertexte, qui est utilisé pour relier une page à l’autre.
L’attribut le plus important de la balise <a> est l’attribut href, qui indique la destination du lien.

Par défaut, les liens apparaissent dans tous les navigateurs suivant cette ordre :
Un lien non visité est souligné et bleu, un lien visité est souligné et violet, Un lien actif est souligné et rouge. Vous pouvez changer le style de vos liens grâce au CSS (Feuille de style).

Les attributs: <hreflang>, <media>, <rel>, <target>, et <type> ne peuvent être utiliser sans l’attribut <href>.

Evolution par rapport au HTML 4.01

En HTML 4.01, la balise <a> pouvait soit être un lien hypertexte soit un lien vers une ancre.
En HTML5, cette balise sera uniquement un lien hypertexte, mais si elle n’a pas l’attribut href, elle reste un espace réservé pour un lien hypertexte.

HTML5 a quelques nouveaux attributs, et certains attributs HTML 4.01 ne sont pas pris en charge dans HTML5.

Attribut pour la balise : <a>

href : url .. Indique l'adresse de la page vers laquelle pointe le lien

hreflang : language code ... Indique la langue de la page vers laquelle pointe le lien

media : media query ...
Indique le type de document vers laquelle pointe le lien. La valeur par défaut est : all

rel : alternate, author; bookmark; external; help; license; next; nofollow; noreferrer; prefetch; prev; search; sidebar; tag ...
Indique la relation entre le document actuel et celui vers lequel pointe le lien

target : _blank; _parent; _self; _top; framename ... Indique l'emplacement d'ouverture du lien

type : mime_type ... Indique le Type MIME du document vers lequel pointe le lien

Les balises de navigation d'un document HTML5

<a> : Définit un lien hypertexte

<link> : Définit la relation entre un document et une ressource externe
( par exemple une feuilles de style )

<nav> : Définit des liens de navigation
