La Balise Fieldset

Avec la balise <fieldset> vous pouvez grouper différentes partie d’un formulaire.
Selon le navigateur, les différents groupes seront séparés par des lignes ou des boites (encadré) à l’affichage.
La balise <legend> définit une légende pour un groupe d’élément contenu dans la balise <fieldset>.

• Le <fieldset> a plusieurs attributs :

• Disable : Spécifie qu’un groupe d’éléments relier du formulaire doit être désactivé
• form : Indique si l’élement <fieldset>; appartient à un ou plusieurs formulaires
• name : Indique le nom du groupe d’élément <fieldset>;
