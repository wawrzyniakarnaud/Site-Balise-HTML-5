BALISE LINK

La balise <link> définit la relation entre un document et une ressource externe.
Elle est surtout utilisée pour lier les feuilles de style CSS et se place dans l’élément <head>

Exemple:

attributs:

-href= spécifie la destination du lien<
-hreflang= spécifie la langue du document dans le lien
-media= spécifie le type de média pour lequel le lien est optimisé
-rel= spécifie la relation entre le document courant et le document lié
-sizes= spécifie la taille des icônes pour les médias visuels
-type= précise le type mine du document lié
